Linter for CudaLint plugin.
Supports Rust lexer.

You must have "rustc" installed on your system. To install rustc, install Rust https://www.rust-lang.org/

Linter has options in linter.py, did not test in Cudatext version, don't know Rust:
    use_cargo = False
    use_cargo_check = False
    use_crate_root = False
    cargo_config = None
    crate_root = None


Ported from SublimeLinter-rustc by Alexey T.
